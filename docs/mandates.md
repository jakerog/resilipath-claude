## 1. PLATFORM MANDATE

**ResiliPath** is a multi-tenant, subscription-based, modular SaaS platform that enables organizations of any industry and size to plan, execute, rehearse, and continuously improve their Business Continuity Management (BCM) and Disaster Recovery (DR) programs.

The platform is built entirely by AI, hosted on GCP free-tier services and Vercel, and targeted at a **May 1, 2026** MVP launch with GPI as the first customer.

---

## 2. NON-NEGOTIABLE MANDATES

### 2.1 Architecture Mandates
- **M-ARCH-001:** Multi-tenant. Every tenant's data is strictly isolated at the Firestore collection level.
- **M-ARCH-002:** Cloud-native on **GCP + Firebase**. Free tier maximized at every layer.
- **M-ARCH-003:** All data encrypted in transit (HTTPS/TLS enforced). Firebase handles encryption at rest.
- **M-ARCH-004:** Modular architecture. Each feature domain is a discrete Cloud Run service or Firebase Function.
- **M-ARCH-005:** REST API first. All features have API endpoints before UI is built.
- **M-ARCH-006:** Immutable audit log. All data mutations produce an append-only audit record.
- **M-ARCH-007:** **No Kubernetes. No self-managed infrastructure.** Serverless-first: Cloud Run + Firebase Functions.

### 2.2 Stack Mandates
- **M-STACK-001:** **Backend runtime:** Node.js 22 + TypeScript on **GCP Cloud Run** (serverless, free tier: 2M requests/month)
- **M-STACK-002:** **Primary database:** **Cloud Firestore** (Native mode, free tier: 50K reads/day, 20K writes/day, 1GB)
- **M-STACK-003:** **Authentication:** **Firebase Authentication** (free, unlimited users, email/password + Google SSO)
- **M-STACK-004:** **Real-time updates:** **Firebase Realtime Database** (free tier: 1GB storage, 10GB/month transfer)
- **M-STACK-005:** **Frontend framework:** **React 18 + TypeScript + Vite** on **Vercel** (free hobby tier)
- **M-STACK-006:** **File storage:** **Firebase Storage** (free tier: 5GB storage, 1GB/day download)
- **M-STACK-007:** **Background jobs:** **GCP Cloud Tasks** (free tier: 1M tasks/month)
- **M-STACK-008:** **Email (MVP):** **Firebase Extensions — Trigger Email** (uses SendGrid free tier: 100 emails/day) or **Resend** free tier (100 emails/day)
- **M-STACK-009:** **IaC:** **Terraform** with GCP provider — all infrastructure as code, AI-maintained
- **M-STACK-010:** **CI/CD:** **GitHub Actions** (free tier for public; 2,000 min/month private)
- **M-STACK-011:** **API framework:** **Hono.js** (lightweight, TypeScript-native, runs on Cloud Run and edge — replaces Fastify given serverless constraint)

### 2.3 Product Mandates
- **M-PROD-001:** DR Exercise Manager is the sole MVP module. Nothing else ships May 1, 2026.
- **M-PROD-002:** Task statuses: `not_started`, `in_progress`, `completed`, `failed`, `delayed`, `optional`, `cancelled`.
- **M-PROD-003:** RBAC: Admin, Moderator, User, Report — enforced at API and Firestore security rules level.
- **M-PROD-004:** DR Bridge mode is required in MVP — dedicated full-screen live view for exercise calls.
- **M-PROD-005:** PWA required — installable, mobile-native feel, works on phone during exercise.
- **M-PROD-006:** Skeuomorphic / dark operational UX design language.
- **M-PROD-007:** Report export: PDF and XLSX.
- **M-PROD-008:** Resource Check-In deferred to Phase 2.
- **M-PROD-009:** SMS deferred to Phase 2+.

### 2.4 Cost Mandates
- **M-COST-001:** MVP infrastructure cost = **$0/month** on GCP + Firebase + Vercel free tiers.
- **M-COST-002:** Every service's free tier limit is documented. Alert triggers at 70% consumption.
- **M-COST-003:** No paid third-party services in MVP. Exception: Resend free tier (100 emails/day) if needed for email.

### 2.5 Launch Mandate
- **M-LAUNCH-001:** ResiliPath MVP is live by **May 1, 2026**.
- **M-LAUNCH-002:** GPI can create, run, and report on a complete DR exercise at launch.
- **M-LAUNCH-003:** Demo environment pre-seeded with real GPI exercise data from analyzed Excel runbook.
