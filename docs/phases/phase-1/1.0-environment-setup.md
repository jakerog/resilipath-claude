# Sub-Phase 1.0 — Environment Setup
**Version:** 0.2.0  
**Date:** 2026-03-29  

**Sub-Phase Title:** Environment Setup  
**Primary Concern:** Provision all accounts, CLI tools, and project scaffolding required before any code can be written. This is a one-time prerequisite checklist — not a recurring phase. Every item produces a concrete, verifiable output. No code is written here.  
**Dependencies:** None. This is the absolute starting point.  

---

## Overview

Because ResiliPath is built entirely by AI with zero human engineering infrastructure in place, this sub-phase establishes the foundation from scratch: accounts, CLI tools, repository, project structure, and CI/CD wiring. An AI coding agent cannot begin Sub-Phase 1.1 until every item here is complete.

---

## Atomic Task List

### 1.0.A — Account Setup

- [ ] **Task 1.0.A.1:** Create **Google Cloud Platform** account at `console.cloud.google.com`
  - Use a dedicated Google account for ResiliPath (not personal)
  - Enable billing (required even for free tier — no charges at free tier usage)
  - Record: `GCP_PROJECT_ID`, `GCP_BILLING_ACCOUNT_ID`

- [ ] **Task 1.0.A.2:** Create **Firebase project** linked to the GCP project
  - Go to `console.firebase.google.com` → Add project → Link to GCP project created above
  - Project name: `resilipath-prod`
  - Enable Google Analytics: No (not needed for MVP)
  - Record: `FIREBASE_PROJECT_ID`

- [ ] **Task 1.0.A.3:** Create **GitHub** account/organization for ResiliPath
  - Create organization: `resilipath-app` (or equivalent)
  - Create repository: `resilipath` (private, with README)
  - Record: `GITHUB_REPO_URL`

- [ ] **Task 1.0.A.4:** Create **Vercel** account at `vercel.com`
  - Sign up with the GitHub account from 1.0.A.3
  - Record: `VERCEL_TOKEN` (from Account Settings → Tokens)

- [ ] **Task 1.0.A.5:** Create **Resend** account at `resend.com` (email sending)
  - Free tier: 100 emails/day
  - Create API key
  - Record: `RESEND_API_KEY`

- [ ] **Task 1.0.A.6:** Register domain `resilipath.app` (or preferred domain)
  - Recommended registrar: Cloudflare Registrar (~$10/year)
  - Point nameservers to Vercel (for frontend) and to GCP Cloud Run custom domain (for API)
  - Record: `DOMAIN_NAME`

### 1.0.B — GCP Services Activation

- [ ] **Task 1.0.B.1:** Enable required GCP APIs via `gcloud services enable`:
  ```bash
  gcloud services enable \
    run.googleapis.com \
    cloudtasks.googleapis.com \
    cloudbuild.googleapis.com \
    secretmanager.googleapis.com \
    artifactregistry.googleapis.com \
    firebase.googleapis.com \
    firebasehosting.googleapis.com
  ```

- [ ] **Task 1.0.B.2:** Create GCP Artifact Registry repository for Docker images
  ```bash
  gcloud artifacts repositories create resilipath \
    --repository-format=docker \
    --location=us-central1
  ```
  Record: `ARTIFACT_REGISTRY_URL=us-central1-docker.pkg.dev/{PROJECT_ID}/resilipath`

- [ ] **Task 1.0.B.3:** Create GCP Secret Manager secrets for all sensitive values:
  - `FIREBASE_SERVICE_ACCOUNT_KEY` (JSON, from Firebase Admin SDK)
  - `RESEND_API_KEY`
  - `FIREBASE_WEB_API_KEY`
  - `REALTIME_DB_URL`
  - Verify: `gcloud secrets list` shows all 4 secrets

- [ ] **Task 1.0.B.4:** Create GCP Service Account for Cloud Run API service
  ```bash
  gcloud iam service-accounts create resilipath-api \
    --display-name="ResiliPath API Service Account"
  ```
  Grant roles: `roles/datastore.user`, `roles/secretmanager.secretAccessor`, `roles/cloudtasks.enqueuer`

- [ ] **Task 1.0.B.5:** Create GCP Service Account for GitHub Actions CI/CD
  ```bash
  gcloud iam service-accounts create resilipath-cicd
  ```
  Grant roles: `roles/run.admin`, `roles/artifactregistry.writer`, `roles/iam.serviceAccountUser`
  Download key JSON → add as GitHub Secret `GCP_SA_KEY`

### 1.0.D — Terraform Setup

- [ ] **Task 1.0.D.1:** Install Terraform CLI (`brew install terraform` or equivalent)

- [ ] **Task 1.0.D.2:** Create `infrastructure/terraform/` directory in repository

- [ ] **Task 1.0.D.3:** Write `infrastructure/terraform/main.tf` — provider configuration:
  ```hcl
  terraform {
    required_providers {
      google = { source = "hashicorp/google", version = "~> 5.0" }
      google-beta = { source = "hashicorp/google-beta", version = "~> 5.0" }
    }
    backend "gcs" {
      bucket = "resilipath-terraform-state"
      prefix = "prod"
    }
  }

  provider "google" {
    project = var.project_id
    region  = var.region
  }
  ```

- [ ] **Task 1.0.D.4:** Create GCS bucket for Terraform state storage (manual, one-time):
  ```bash
  gsutil mb -p {PROJECT_ID} -l us-central1 gs://resilipath-terraform-state
  gsutil versioning set on gs://resilipath-terraform-state
  ```

- [ ] **Task 1.0.D.5:** Write `infrastructure/terraform/variables.tf` with all required variables:

### 1.0.F — CI/CD Pipeline

- [ ] **Task 1.0.F.1:** Create `.github/workflows/ci.yml` — runs on every PR:
  ```yaml
  jobs:
    ci:
      steps:
        - checkout
        - setup-pnpm
        - pnpm install --frozen-lockfile
        - pnpm turbo typecheck
        - pnpm turbo lint
        - pnpm turbo test
  ```

- [ ] **Task 1.0.F.2:** Create `.github/workflows/deploy.yml` — runs on push to `main`:
  ```yaml
  jobs:
    deploy-api:
      steps:
        - checkout
        - authenticate to GCP via service account key
        - pnpm turbo build --filter=@resilipath/api
        - docker build -t $ARTIFACT_REGISTRY_URL/api:$SHA apps/api
        - docker push $ARTIFACT_REGISTRY_URL/api:$SHA
        - gcloud run deploy resilipath-api --image=$ARTIFACT_REGISTRY_URL/api:$SHA

    deploy-web:
      steps:
        - pnpm turbo build --filter=@resilipath/web
        - vercel --prod (via Vercel GitHub integration — auto-deploys)

    deploy-firebase:
      steps:
        - firebase deploy --only firestore:rules,firestore:indexes,database,storage
  ```

- [ ] **Task 1.0.F.3:** Add all required GitHub Secrets:
  - `GCP_SA_KEY` (service account JSON for CI/CD)
  - `VERCEL_TOKEN`
  - `VERCEL_ORG_ID`
  - `VERCEL_PROJECT_ID`
  - `FIREBASE_TOKEN` (from `firebase login:ci`)

- [ ] **Task 1.0.F.4:** Configure Vercel project: link to GitHub repo, set environment variables, set framework preset to Vite

- [ ] **Task 1.0.F.5:** Create `apps/api/Dockerfile`:
  ```dockerfile
  FROM node:22-alpine AS builder
  WORKDIR /app
  RUN corepack enable pnpm
  COPY pnpm-lock.yaml package.json pnpm-workspace.yaml ./
  COPY packages/ packages/
  COPY apps/api/ apps/api/
  RUN pnpm install --frozen-lockfile
  RUN pnpm turbo build --filter=@resilipath/api
  
  FROM node:22-alpine
  WORKDIR /app
  COPY --from=builder /app/apps/api/dist ./dist
  COPY --from=builder /app/apps/api/package.json .
  COPY --from=builder /app/node_modules ./node_modules
  EXPOSE 8080
  CMD ["node", "dist/index.js"]
  ```

- [ ] **Task 1.0.F.6:** Verify end-to-end: push a trivial commit to `main` → CI passes → Cloud Run deploys → `GET /health` returns 200
